knn_regress

Function to determine k nearest neighbors

Syntax

y_pred = knn_regress(x, y)
y_pred = knn_regress(x, y, k)

Parameters

x:

n x m matrix of input parameters, each column represents a different parameter

y:

n x p matrix of target values, each column represents a different variable

y_pred:

n x p matrix of predicted target values.

k:

integer describing number of nearest neighbors, default = 5
