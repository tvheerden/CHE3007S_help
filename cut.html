cut

A function to cut data into bins

Syntax

[bin_assigned, bin_assigned_s, bins] = cut(data, 'nbins', n)
[bin_assigned, bin_assigned_s, bins] = cut(data, 'bins', cut_line)

Parameters

bin_assinged:

vector of values indicating which bin the corresponding data point belongs to

bin_assigned_s:

vector of strings indicating which bin the corresponding data point belongs to

bins:

vector of strings describing the bins

cut_line:

vector of values describing the locations of the cuts

data:

column vector of values to be cut into bins

n:

integer value, describing the number of bins, if 'nbins' specified
